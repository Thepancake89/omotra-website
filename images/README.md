# Images Folder

## Required Images

### og-image.png (Social Sharing)
- **Dimensions:** 1200 x 630 pixels
- **Purpose:** Displayed when sharing links on social media (Facebook, LinkedIn, Twitter/X)
- **Recommendation:** Create an image featuring:
  - Omotra logo/brand
  - Tagline: "Built well. Looked after."
  - Dark teal background (#0A3D42) with coral accent (#E07A5F)

### apple-touch-icon.png (iOS Home Screen)
- **Dimensions:** 180 x 180 pixels
- **Purpose:** Icon when users add site to iOS home screen
- **Recommendation:** Use the diamond logo mark on dark background

### favicon-32x32.png & favicon-16x16.png
- **Purpose:** Browser tab icons (optional - current SVG favicon works well)

## Current Status
The site currently uses an inline SVG favicon which works for most purposes. 
Add og-image.png as a priority for better social sharing appearance.
